Instructions

To select/unselect an element: click it
Add Alignment button will create an alignment button. (Shortcut: double-click an element)
Clicking an alignment button will color its alignment. (double-click to delete)
Download button allows you to download all alignments as a text file.
Load button allows you to load a a previous alignment file.
Show All button highlights all alignments to show their coverage. (Shortcut: ctrl + x)

Alignments

Each element has a unique identifier.
AMR nodes: a, b, a2, b2, x0, x1, ...
AMR edge from a to b: a_ARG0_b, a_mod_b, ...
Word in sentence: 0, 1, 2, ...
Use # to add a comment to an alignment.
